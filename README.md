# 🏥 MedTriage AI

An AI-powered medical triage system that helps prioritize patients based on their symptoms. The application assists healthcare professionals by analyzing patient information and suggesting urgency levels to support faster decision-making.

## Features

* AI-based symptom analysis using Google Gemini
* Patient priority classification
* Patient registration and management
* Dashboard to manage patient records
* Authentication and protected routes
* Responsive user interface

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* React Router
* Context API
* Google Gemini API
